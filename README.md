# Sai Chappidi — Portfolio

Personal portfolio website for **Sai Chappidi**, Computer Science & Engineering student at The Ohio State University.

## Features

- Responsive single-page layout (About, Experience, Projects, Skills, Contact)
- Interactive terminal with navigation commands
- Command palette (`Ctrl+K` / `⌘K`)
- Multiple color themes (Aurora, Terminal Green, Salesforce Blue, Synthwave)
- Scroll animations, project cards, and downloadable résumé

## Tech stack

- HTML5
- CSS3
- Vanilla JavaScript (no build step required)

## Local development

From the project folder:

```powershell
cd e:\site
python -m http.server 8000
```

Then open **http://localhost:8000** in your browser.

You can also open `index.html` directly in a browser.
